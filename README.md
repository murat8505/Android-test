# Android-test

Задание

Необходимо реализовать приложение под Android поглощающее простейший API и выводящее список полученных элементов. По нажатию на элемент необходимо отображать дополнительную информацию.



Требования:

API endpoint - http://www.mocky.io/v2/5a488f243000004c15c3c57e;
оформление и компоновку элементов интерфейса автор выбирает сам: в наличии должен быть прокручиваемый список и дополнительная информация отображаемая по нажатию на элемент данного списка;
SDK и сторонние библиотеки - любые, по личному предпочтению.
Скачать apk файла <a href="https://github.com/murat8505/Android-test/raw/master/apk/android_test.apk">здесь</a>


Возможные вариации

более сложная структура обрабатываемых данных (объект в объекте);
пагинация;
дополнительные функции над элементами списка (что придумается).
